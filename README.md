# Customer Decisioning Lab

Sistema de próxima melhor ação com inferência causal, otimização sob restrições,
experimentação segura e copiloto de IA auditável.

> Status: construção em andamento. Nenhum resultado financeiro deste repositório
> representa impacto real de uma empresa.

## Problema

Escolher, para cada cliente elegível, cashback, pontos, parcelamento ou nenhum
contato, buscando valor incremental em 90 dias e respeitando orçamento,
consentimento, capacidade e frequência.

## Dados

- Benchmark metodológico: Criteo Uplift Prediction Dataset.
- Aplicação empresarial: simulador sintético de clientes de cartões.

## Princípio

O modelo estima. O otimizador recomenda. O experimento prova. O humano aprova.
